# Recommended_Tourist_Attractions_Search_system
## 題目主題
地址 / 地標 → 推薦景點查詢系統 整合 OpenTripMap API 與台灣在地景點資料，實作輸入任意地址或地標，即可查詢周邊推薦景點的互動式系統，作為旅遊建議工具。

## 分析目標與指標
* 預計設計一個輸入以下任一資料後：
1. 中文地址
2. 英文地址
3. 知名景點名稱  
* 能夠帶出以下附近景點資料，並視覺化呈現於地圖上：
1. 地點
2. 類型
3. 評分
4. 維基或知名部落格連結等

## 預期成果
* 支援輸入中文地址/地標，自動查詢附近景點的互動式系統
* 提供地圖視覺化結果，含景點名稱、距離、類型、簡介
* 若是牽涉到住宿問題的話，能將多個訂房網站中的同一飯店價位比較彙整成一個表格，一目瞭然
